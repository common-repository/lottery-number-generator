 Lottery Number Generator
 ! Do not use the shortcode & widget at a time.
 

== Description ==

[eu-lottery number="5" highest="60" position="none"]

Position: float_left, float_right, middle, left, right


output: 5 integer values between 1 & 60 
restrictions:   highest can not be less then number


**This plugin is a sample on how to create a plugin which:**
* provides an option screen,
* saves some options,
* use jQuery,
* use Ajax,
* replaces a tag.

By the way, **it generates lotteries numbers** 

If you need more functionalities, feel free to ask (email)..


== Supported languages ==

* English

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload folders and files  to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place '[eu-lottery]' in a post (if you haven't changed the tag caption in the options). Or you can use the widget.
